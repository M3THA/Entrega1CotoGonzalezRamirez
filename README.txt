Guia para navegar en nuestro proyecto.

  1) Iniciar la consola de VSC en el directorio del proyecto.
  2) Ejecutar el comando python manage.py runserver en la consola.
  3) Acceder al link que devuela la consola.
  4) Dentro del link, tipear en el buscador la siguiente URL: http://127.0.0.1:8000/App/ para visualizar las urls navegables.

 Listado de Urls:
 
 App/inicio/ ---> Pagina principal.
 App/usuario/ ---> Envia a la pagina para agregar usuarios desde el formulario.
 App/producto/ ---> Envia a la pagina para agregar productos desde el formulario.
 App/descuento/ ---> Envia a la pagina para agregar descuentos (Los descuentos no estan creados todavia para operarse con calculos).
 App/buscar/Producto/ ---> Envia a la pagina donde se encuentra el formulario para buscar los productos.
 App/buscar/ ---> Envia a la pagina del resultado solicitado.
